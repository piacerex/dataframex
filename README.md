# Dataframex

[Dataframex](https://hex.pm/packages/dataframex) is a Elixir data manipuration utilities like Python's pandas.

See the [online documentation](https://hexdocs.pm/dataframex).

## Installation

Add to your ```mix.exs``` file:

```elixir
def deps do
  [
    { :dataframex, "~> 0.5" }
  ]
end
```
